#BIENVENIDOS

El archivo index.php es el encargado de inicializar la aplicación, cargar los recursos necesarios y mostrar la vista principal del sistema, desde donde el usuario puede navegar por las diferentes funcionalidades del proyecto.

Fue inicializado en esta ruta: http://localhost/curso_php/Aprende_php/proyecto/php/index.php 

Aunque la idea es que inicie en cualquier ruta manejado en php, algo recomendado es que maneje proyecto/php/ para que abra directamente el index de php

El archivo index.php cumple con las siguientes responsabilidades:

Iniciar la ejecución del sistema en entorno local.

Establecer la conexión con la base de datos.

Cargar archivos de configuración y funciones auxiliares.

Obtener y mostrar dinámicamente la información almacenada (entradas, reseñas o contenidos).

Renderizar la interfaz principal utilizando HTML, CSS y Bootstrap.

Gestionar la navegación hacia vistas secundarias (detalle de entradas, categorías, login, etc.).
